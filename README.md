# PrGo
## Unificação do Transporte Público Paranaense via Digitalização

---

## O Problema
Cidadãos paranaenses utilizam sistemas desconectados para transporte municipal e intermunicipal, gerando múltiplos cartões e processos de recarga complexos.

---

## A Solução
Uma plataforma digital que centraliza:

- Recarga única com múltiplas formas de pagamento  
- Integração automática entre sistemas municipais  
- Histórico de viagens e gestão de saldo  

---

## Arquitetura (em definição)

- Mobile: React Native / Flutter (pendente)  
- Backend: Java Spring / Node.js (pendente)  
- Padrões: Clean Architecture, Repository Pattern  
- Metodologia: Engenharia de Software II — 2026/1
